# SolarFlare

SolarFlare is a curated archive of AI assistant prompts, system prompts, tool schemas, and related prompt artifacts from coding agents, app builders, research assistants, and other AI products.

The repository is organized by product or provider so the materials are easy to browse, compare, and study.

## What's Inside

- Prompt and system-message text files
- Tool and action schemas in JSON or similar formats
- Product-specific notes, examples, and extraction references
- Open source prompt references for comparison

## Repository Layout

Each top-level folder generally represents one product, provider, or prompt family. Examples include:

- `Anthropic/`
- `Cursor Prompts/`
- `Google/`
- `Open Source prompts/`
- `v0 Prompts and Tools/`
- `VSCode Agent/`
- `Windsurf/`

Some folders contain both prompts and tool definitions. A few include local notes or supporting images when those help explain how the artifact was obtained or interpreted.

## How To Use

Browse the folders directly, or search for a product name, model name, tool name, or behavior you want to compare.

Useful patterns to look for:

- How different agents define goals, constraints, and autonomy
- How tool schemas are shaped for coding, browsing, editing, and planning
- How products structure system prompts for safety, task execution, and user collaboration
- How open source agents differ from commercial products

## Contributing

Contributions are welcome. When adding new material:

- Place files under a clear product or provider folder.
- Prefer descriptive filenames.
- Preserve original formatting where practical.
- Add short notes when provenance, version, date, or extraction method matters.
- Avoid including secrets, private user data, access tokens, or account-specific information.
